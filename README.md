## QRCARD - Client

This repository concerns the client part of the qrcard project.

### Prerequisites

Install all modules
  ```sh
  npm install or yarn
  ```

### Start the project 

```sh
expo start
```

### Commands

1. Reload the app
```sh
r
```
2. Launch on IOS
```sh
i
```
2. Launch on Android
```sh
a
```
